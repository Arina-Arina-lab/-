Основные технологии

Современный сайт строится на трёх "китах": HTML, CSS и JavaScript.

HTML (HyperText Markup Language)

Язык разметки для создания структуры страницы (заголовки, абзацы, списки).

CSS (Cascading Style Sheets)

Язык стилей для оформления внешнего вида (цвета, шрифты, отступы).

JavaScript

Язык программирования для добавления интерактивности (анимация, обработка событий).

Примеры языков и технологий:

  • HTML5 — стандарт разметки.
  • CSS3 — современные стили.
  • JavaScript (ES6+) — логика на клиенте.
  • PHP / Python / Node.js — серверная часть (backend).
  • SQL — работа с базами данных.
